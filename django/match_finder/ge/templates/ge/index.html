A fim, de conhecemos os gostos dos nossos usuários, e assim oferecer os melhores conteúdos voltados para cada público, os categorizamos por genêros (Masculino e Feminino) e fizemos a seguinte pergunta: "Podemos usar o top 3 dos esportes favoritos, para trazer assunto relacionados a esses esses esportes? Esse rank se mantém para cada um dos genêros? ".

Levando em conta nosso banco de dados dos usuários, chegamos ao seguinte gráfico, que aponta o rankeamento geral dos esportes favoritos.

Dado o gráfico chegamos ao top 3 geral dos esportes favoritos:

  1. Futsal

  2. Futebol

  3. Handebol


Analisando os dados para descobrir os esportes favoritos do público feminino, temos o seguinte gráfico:

Com top 3:

  1. Basquete

  2. Vôlei

  3. handebol

Fazendo o mesmo, porém, para o público masculino, temos o seguite gráfico:

Com top 3:

  1. Futsal

  2. Handebol

  3. Futebol

Logo, pela análise dos dados, chegamos à conclusão que não podemos levar em consideração o rankeamento geral dos esportes favoritos para trazer notícias sobre esses esportes aos nossos usuários. Explico: apesar que o top 3 dos esportes favoritos do público masculino serem os mesmos do rakeamento geral, uma parcela muito grande do público feminino não se sentiria atraída e acolhida para acessar a página.
Dessa forma, se queremos que todos se sintam aceitos e acolhidos pelo "Match Finder", devemos sempre ir em busca dos melhores conteúdos e, claro, atender os gostos de cada usuário. Afinal, cada pessoa é única.
